# 🌐 Ficius Juris Club

**Ficius Juris Club** est un site web développé pour un club universitaire dédié à la **vulgarisation et à la promotion du droit**.  
Le site permet de présenter le club, publier des actualités, accueillir de nouveaux membres et gérer les activités via une interface d’administration.

---

## ✨ Fonctionnalités principales

- ✅ Page d’accueil présentant le club et sa mission
- 📰 Section "Actualités" pour partager des événements ou annonces
- 📝 Formulaire d’adhésion pour les nouveaux membres
- 🔐 Interface d’administration sécurisée :
  - Gestion des membres
  - Publication d’actualités
  - Suivi des activités

---

## ⚙️ Technologies utilisées

- **Frontend** : HTML + Tailwind CSS
- **Backend** : PHP
- **Base de données** : MySQL (via phpMyAdmin)

---

## 🚀 Installation locale

1. **Installer** [XAMPP](https://www.apachefriends.org/index.html) ou [WAMP](https://www.wampserver.com/)
2. **Lancer XAMPP/WAMP**  
   👉 Vérifiez que **Apache** et **MySQL** sont activés
3. **Importer la base de données** dans phpMyAdmin :
   - Nom de la base de données : `ficius`
   - Fichier : `ficius.sql`
4. **Placer le dossier du site** (`ficius/`) dans :
   - `htdocs` (pour XAMPP)
   - ou `www` (pour WAMP)
5. **Accéder au site** :
   - Ouvrir votre navigateur et taper : `http://localhost/ficius`
6. **Accéder à l’administration** :
   - URL : `http://localhost/ficius/admin`
   - Mot de passe par défaut : `admin`

---

## 📌 Remarques

- Ce site est pensé pour une utilisation locale ou en ligne sur un serveur supportant PHP et MySQL.
- Il peut être adapté pour d'autres clubs ou associations juridiques.
