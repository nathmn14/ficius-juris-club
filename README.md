# ficius-juris-club
Création du Site Ficius Juris Club, un club dédié à la vulgarisation et à la promotion du droit. Le site comprend une page d'accueil, une section actualité et un formulaire pour les nouveaux memnbres. Une interface d'administration a été aussi fournie afin de gérer les activités du club et les membres, ainsi qu'alimenter l'actualité sur le site.
